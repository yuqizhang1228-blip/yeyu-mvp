> **原则**：P0 抽屉 **以当前代码为准** 验收与迭代；对齐 Figma 0515 时 **只改样式，不擅自按 YUQ-30 全文扩功能**。  
> 下文「设计有、代码暂无」**不是产品否决**，而是 **现阶段未做 + 避免 Agent 误加**；要上哪一项，在 Linear 单独立项后再改本文。

## 触发与壳层

| 项 | 实现 |
|----|------|
| 打开 | 首页 / 对话页顶栏左侧菜单；`appState.drawerOpen = true` |
| 关闭 | 点遮罩、选导航项、选历史会话后自动关 |
| 动画 | 自左滑入，宽约 **304pt**，遮罩 `YeyuColor.overlay` |
| 挂载 | `HomeView`、`ChatView` 内各一份 `SideDrawerView`；对话页传入 `currentSessionId` 用于高亮 |

## 面板内容（页眉 / 会员 Banner / 入口 / 最近对话）

### 1. 页眉

- 标题：**夜屿**
- 副文案：当前为「一款情绪拆解 AI 助手」

### 1.5 会员 Banner（2026-06-03 加入 · 226:2407）

- 暗底 `#282828` + 月/山插画（原生绘制）+「成为会员」
- 点击：暂占位「敬请期待」；**二级会员转化弹窗待 YUQI 后续补**（届时把 banner 的 action 接到该弹窗）

### 2. 固定入口（4 个，2026-06-03 起对齐设计稿 226:2399）

| 入口 | icon | 行为 |
|------|------|------|
| **行动卡片** | heart | 关抽屉 → `openHistory()` → `HistoryView`（已保存卡片列表）。命名沿用 App 术语，未改为稿子的「心情卡片」（YUQI 裁决） |
| **个性化** | square.and.pencil | 关抽屉 → `openPersonalization()` → 个性化 sheet（YUQ-37） |
| **支持鼓励** | tag | **占位**：暂无 PRD/路由，点击弹「敬请期待」轻提示、不关抽屉（YUQI 2026-06-03 裁决）。接真实页面需先立项 |
| **设置** | gearshape | 关抽屉 → `openSettings()` → `SettingsView` |

### 3. 最近对话

- 标题：**最近对话**
- 数据：SwiftData `ChatSession`，按 `updatedAt` 倒序，最多 **12** 条
- 展示：`session.title`（含 AI 生成标题逻辑，见会话服务）
- 点击：关抽屉；若已在当前会话则 no-op；若在对话页则 `replaceChat(sessionId:)`，否则 `openChat(sessionId:)`
- 当前会话：文案色 `YeyuColor.primary`
