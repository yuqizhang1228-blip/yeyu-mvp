# iOS v1.1 本地构建说明（未推 GitHub）

## 本批新增

| 能力 | 说明 |
|------|------|
| **SSE 流式对话** | `api/chat.js` 支持 `stream: true`；iOS `ChatAPIClient.sendStream` + 对话页逐字显示 |
| **流式回退** | 线上若仍是旧 API，iOS 自动改走非流式 `send` |
| **本地记忆** | `UserProfileService` 对齐 H5 `yeyu_profile`；Chip 带摘要；设置里可写备忘 |
| **保存卡片记念头** | 写入 `recentCardTopics` 供下次 Chip |

## 本地联调流式

```bash
# 终端 1：必须用更新后的 api/chat.js
cd yeyu-mvp
export DASHSCOPE_API_KEY="..."
npm run dev

# 终端 2：iOS 把 ChatAPIClient.baseURL 改为 http://127.0.0.1:3000/api/chat
# 模拟器可用 Mac 局域网 IP 替代 127.0.0.1
```

生产 `yeyu-mvp.vercel.app` **在你未部署前**不会流式，App 仍可用（回退整包）。

## 验收要点

1. 本地 API + 改 baseURL → 发消息后可见逐字输出  
2. 设置写「本地记忆」→ 回首页刷新 Chip，场景更贴个人  
3. 保存一张卡片 → 再次打开首页，Chip 可贴近该念头主题
